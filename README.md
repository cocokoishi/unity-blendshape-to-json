**Unity blendshape to json tool for vrchat?**

It is a simple tool to export your mesh's blendshape to clean Json.

**WTF can it do?**

By converting blendshapes to JSON, you can feed that data directly into an LLM (Large Language Model). This allows an AI to:

* **Analyze Facial Logic:** Let an AI help you decipher how an avatar's face functions, which is particularly useful for poorly optimized avatars cluttered with redundant or "junk" blendshapes.
* **Data Cleanup:** Identify and categorize blendshapes without manual trial-and-error in the Unity Inspector.

**AI-driven Avatar Face Shaping**

* **AI-Driven Shaping:** Use an LLM to generate new JSON configurations to programmatically reshape the avatar's face or automate complex expressions.  
* Of course, you'll need to manually copy the data out for LLM processing and then paste it back in afterward.

**How to install**

Download the .cs file and drop it into your project.
Make sure the script is inside an Editor folder. example:Assets/Editor/BlendShapeDataEditor.cs
